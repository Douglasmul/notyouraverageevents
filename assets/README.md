# Assets Directory

## Featured Event Image

To replace the placeholder event flyer with your own screenshot:

1. Save your screenshot as `event-flyer.png` or `event-flyer.jpg` in this directory
2. Update the image source in `src/app.js` if using a different filename
3. The image will be displayed prominently at the top of event modals

The current placeholder is `event-flyer.svg` which can be replaced with your provided screenshot:
https://github.com/user-attachments/assets/a332d738-b1c7-4356-a410-cf130c2e5f90

The image should be optimized for web display and will be responsively sized to fit the modal width.
